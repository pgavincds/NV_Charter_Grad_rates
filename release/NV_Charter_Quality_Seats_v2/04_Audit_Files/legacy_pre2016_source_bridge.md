# Legacy Pre-2016 Source Bridge

This note records the narrower historical bridge for extending Nevada charter quality-seats work further back without pretending that every older year already exists in the same modern file structure.

## Confirmed legacy evidence

- Nevada Report Card district/state-charter accountability PDFs are available for the state-charter bucket beginning with the `2012-13` reporting cycle.
- In practical terms, the current bridge looks like this:
  - `2012-13`: legacy state-charter accountability PDF confirmed
  - `2013-14`: legacy state-charter accountability PDF confirmed
  - `2014-15`: legacy state-charter accountability PDF confirmed
  - `2015-16`: modern school-ratings CSV plus enrollment workbook already normalized into this release
  - `2016-17`: modern school-ratings CSV plus enrollment workbook already normalized into this release

## Important governance note

- The state-charter reporting bucket predates SPCSA as a statutory entity under NRS 388A.
- That means pre-`2013` / early-`2013` state-charter evidence should be treated as a historical state-charters bucket rather than back-labeled as if SPCSA already existed in its later legal form.

## What this means for the next pass

- We no longer need to treat the pre-`2015-16` period as an evidence vacuum.
- We do still need a format bridge:
  - older years rely on Nevada accountability PDFs
  - newer years rely on normalized school-ratings CSVs and enrollment workbooks
- The next historical extension should therefore focus on extracting usable seat / rating evidence from the `2012-13` through `2014-15` legacy PDFs, not on cataloging every saved file on disk.
