# SPCSA Cohort Reconciliation Notes, 2013-14 to 2015-16

## Bottom line

The dramatic size drop occurs almost entirely between `2013-14` and `2014-15`, not between `2014-15` and `2015-16`.

- `2013-14` reconstructed SPCSA charter-only cohort: `2,950`
- `2014-15` reconstructed SPCSA charter-only cohort: `922`
- `2015-16` reconstructed SPCSA charter-only cohort: `910`

So:

- `2013-14 -> 2014-15`: `-2,028` cohort, `-889` graduates
- `2014-15 -> 2015-16`: `-12` cohort, `+49` graduates

## Main school-level drivers of the 2013-14 to 2014-15 drop

The decline is heavily concentrated in a few large legacy SPCSA operators:

- `Nevada Virtual Charter`: `1,339 -> 242` (`-1,097`)
- `Beacon Acad`: `555 -> 115` (`-440`)
- `Nevada Connections Acad`: `491 -> 199` (`-292`)
- `Argent Preparatory Academy`: `291 -> 157` (`-134`)
- `Alpine Acad`: `88 -> 40` (`-48`)
- `NSHS Legacy Single-Campus`: `154 -> 111` (`-43`)

The only positive offset in that transition is:

- `CASLV Sandy Ridge`: `0 -> 31` (`+31`)

## Interpretation

The best current reading is:

1. The reconstructed school-level evidence shows that the early SPCSA charter-only graduation universe was dominated by very large online and alternative cohorts.
2. Those cohorts were much smaller by `2014-15`, especially at `Nevada Virtual`, `Nevada Connections`, and `Beacon`.
3. After that initial break, the SPCSA charter-only cohort is relatively stable from `2014-15` to `2015-16`.

## Important caution

`2013-14` remains a high-uncertainty legacy year.

The official Nevada public context file shows:

- `2013-14 SPCSA Official cohort = 922`

But the reconstructed SPCSA charter-only school sum in this release is:

- `2013-14 SPCSA Charter Only cohort = 2,950`

That mismatch is too large to ignore. It suggests at least one of the following:

- early Nevada sponsor-level reporting was not aggregating the same entities now visible in the school-level portal rows
- some legacy school rows were reported differently across interfaces
- early online / alternative rows may have been handled inconsistently in sponsor totals

So the current reconciliation supports a substantive explanation for the drop, but it also indicates that `2013-14` should be treated as a legacy outlier year rather than a clean apples-to-apples baseline.
